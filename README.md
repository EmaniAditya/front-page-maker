# Front Page Maker

Create professional project front pages and export them as A4-sized PDFs.

Live demo → https://emaniaditya.github.io/front-page-maker

## Quick start
```bash
npm install
npm run dev
```

Built with React, Tailwind CSS and jsPDF.
